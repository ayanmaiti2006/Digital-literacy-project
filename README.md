## 📖 Project Overview
This repository serves as a comprehensive Digital Literacy Portfolio created in my role as a **Student Digital Ambassador**. The project is designed to help peers navigate the digital world safely and professionally through five specialized modules.

## 📂 Repository Structure
* **task-1-presentation/**: Contains the Digital Literacy Infographic and design description.
* **task-2-portfolio/**: Includes screenshots of foundational professional profiles (GitHub, LinkedIn, Kaggle).
* **task-3-platforms/**: Contains proof of coding challenges and the Digital Literacy Awareness Quiz.
* **task-4-email-etiquette/**: Features professional email drafts and a social media responsibility checklist.
* **task-5-cybercrime/**: Includes a case study on UPI fraud and an online safety prevention checklist.
* **report/**: The final comprehensive Project Report in PDF format.

## 🛠️ Module Summaries

### Module 1: Visual Communication
I created a visual resource using **Canva** to define digital literacy and highlight essential tools and safety practices for students.

### Module 2: Digital Presence
Established a professional footprint on **GitHub, LinkedIn, and Kaggle**.This ensures a credible online identity for future academic and career opportunities.

### Module 3: Coding & Collaboration
* **Coding:** Completed a beginner-level challenge on **HackerRank** to validate technical logic.
* **Collaboration:** Designed a 5-question awareness quiz using **Google Forms** to engage with my batchmates.

### Module 4: Professional Communication
Drafted formal emails for academic extensions and internship inquiries.I also developed a **Social Media Do's and Don'ts** list to promote responsible digital citizenship.

### Module 5: Cybercrime Awareness
Researched a detailed case study on **UPI/Online Payment Fraud** and compiled a 'Stay Safe Online' checklist featuring the National Cyber Crime reporting channels (1930 and cybercrime.gov.in).
